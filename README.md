# 鲲鹏跃潮

给伍建忠 / 小伍工作室做的离线中文互动开场页，主打第一眼视觉冲击、手机友好、直接双击可跑。

## 亮点

- 纯 `index.html`，无外网依赖，浏览器直接打开
- Canvas 主视觉，包含潮汐、流光、鲲鹏巡游、点击波纹、拖动追踪
- 手机和桌面都能用，触摸和鼠标反馈都很明显

## 运行方式

直接用浏览器打开 [`/Users/xiaowu/Desktop/明早惊艳项目/index.html`](/Users/xiaowu/Desktop/明早惊艳项目/index.html)。

## 长任务经验

- 先把“必须离线可开箱”钉死，再做视觉层叠，避免最后被依赖卡住
- 交互只保留最直观的两种：点击激波、拖动牵引，完成度比功能堆砌更重要
- 移动端优先控制粒子数量和绘制层数，观感和流畅度要一起保
